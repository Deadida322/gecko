# gecko

## Docker

```bash
# run in docker
$ .\docker-scripts\docker-run-client.ps1
# stop and remove docker container
$ .\docker-scripts\docker-down.ps1
```
## NPM

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
